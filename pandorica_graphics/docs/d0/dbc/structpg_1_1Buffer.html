Detailed Description

A struct for memory space on the VRAM.

The buffer is memory space on the VRAM. By default all buffers should be created with the vertex usage flag, only use a different one if you know what you're doing.

If you want to upload data to the buffer, you have to use a TransferBuffer.

Buffers are part of the GPUDevice and can be referenced by transferbuffers.

Note
A vertex and index buffer with correct data is automatically created in CreateStandardContext().

GPUDevice, TransferBuffer, CreateBuffer(), DeleteBuffer(), SetBufferAttributes(), ResizeBuffer(), GetBuffer()
