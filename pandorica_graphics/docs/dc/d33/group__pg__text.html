◆ SetTextAttributes()

bool pg::SetTextAttributes ( const id_t textId,
const std::vector< attributeAndValue > & attributes )

Set the attributes of a text.

Supported attribute(s):
OBJECT_ID, TEXTURE_ID (fontId), (*)POSITION_X, (*)R_POSITION_X, (*)POSITION_Y, (*)R_POSITION_Y, (*)POSITION_Z, (*)R_POSITION_Z, (*)ROTATION_X, (*)R_ROTATION_X, (*)ROTATION_Y, (*)R_ROTATION_Y, (*)ROTATION_Z, (*)R_ROTATION_Z, (*)RED, (*)R_RED, (*)GREEN, (*)R_GREEN, (*)BLUE, (*)R_BlUE, (*)ALPHA, (*)R_ALPHA, (*)ORIGIN_X, (*)R_ORIGIN_X, (*)ORIGIN_Y, (*)R_ORIGIN_Y

(*) = actually modifying the attached object.

To change the string data, you have to use SetText().

Note
Every attribute that works on objects will not throw an error, however they will not change anything about the text.
See also
Text, Object, SetText()
