Detailed Description

A camera creating a scene.

A camera is a point in space that looks at another point in that space. cameras are part of the window.

Note
The camera only sees enabledWorkspaces.
See also
Workspace, CreateCamera(), DeleteCamera(), SetCameraAttributes(), GetCamera()
