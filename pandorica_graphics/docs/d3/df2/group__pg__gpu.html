const SDL_Surface * surface,
SDL_GPUTextureCreateInfo info )

Creates a texture.

This function creates a texture at the specified id with the specified data (surface or path). That data is then copied to the transferbuffer.

Parameters
textureIdThe id at which to create the texture.
transferBufferIdThe id of the transferbuffer that is used for uploading the texture.
surfaceThe image data of the texture (usually acquired with 'IMG_Load()').
infoInfo about the texture. Don't set width and height and if you want to, you can change the other attributes, although you can just leave this empty.
Returns
true if it succeeded, false if it failed. See the logs for more information.
See also
Texture, DeleteTexture()
Parameters
textureIdThe id at which to create the texture.
transferBufferIdThe id of the transferbuffer that is used for uploading the texture.
pathThe relative path at which the texture file is located.
infoInfo about the texture. Don't set width and height and if you want to, you can change the other attributes, although you can just leave this empty.
Returns
true if it succeeded, false if it failed. See the logs for more information.
