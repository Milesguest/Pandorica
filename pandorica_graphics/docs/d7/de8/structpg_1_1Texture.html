Detailed Description

A texture holding image data on the GPU.

A texture is used to store image data on the gpu. It is basically a buffer whose data is interpreted as an image.

To actually use the texture you first have to call CreateTexture(). You can do that with multiple textures and in the end you call UploadTextures().

A texture is part of the GPUDevice and is referenced by transferbuffers and objects.
